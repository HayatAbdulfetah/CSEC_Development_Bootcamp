Personal Profile Mini Project

This is a simple Personal Profile mini project created for the CSEC Development Bootcamp to practice basic HTML and CSS skills.

Technologies Used

* HTML5
* CSS3

## Features

* Personal name and short bio
* Rounded profile image
* Simple contact form (name, email, message)

How to Run

Open `index.html` in any web browser.
